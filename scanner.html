사용 방법

1

Irminsul 다운로드

Releases 페이지에서 최신 버전의 .exe 파일을 받으세요. 다운로드 후 마우스 오른쪽 클릭 → 관리자 권한으로 실행합니다. 자세한 사용법은 공식 문서를 참고하세요.

2

원신 시작과 함께 스캔

Irminsul을 먼저 실행한 후 원신을 시작하면 자동으로 데이터를 캡처합니다.

3

GOOD JSON 내보내기

스캔 완료 후 GOOD 포맷으로 JSON 파일을 내보냅니다.

4

여기서 Import

내보낸 JSON 파일을 위 Import 영역에 업로드하면 데이터가 자동으로 등록됩니다.
